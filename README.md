## 🧩 Understanding the components

While you only see the chat interface, the tool uses several parts to function. These components work behind the scenes to keep your computer tasks running smoothly.

### Local AI orchestration
The tool acts as an orchestrator. It listens to your request, determines which application needs to perform the work, and sends the correct instructions to that application. Because this happens on your machine, your commands do not leave your computer unless you explicitly choose to use an online AI service.

### Integration with Adobe software
The tool uses specialized connections to talk to Photoshop and After Effects. These connections allow the tool to trigger actions that you usually perform with a mouse and keyboard. You do not need to install plugins or extensions yourself; the tool manages these connections when you launch it.

### Docker management
If you use Docker, you know it involves many terminal commands. This tool replaces those commands. It reads your Docker configuration and provides readable status updates in the chat window. You can restart services or check container logs with a single sentence.

## 🔧 Frequently asked questions

### Do I need a coding background?
No. You do not need to know programming. The tool uses plain English to execute your commands.

### Can I stop a command once it starts?
Yes. You can type "Stop" or "Cancel" in the chat window to halt any active task.

### Why is the tool asking for permissions?
Windows requires permission to allow one application to perform actions inside another. The tool asks for these permissions during the start to ensure it can open, edit, and save files on your behalf.

### What if an error occurs?
The chat window will display a message describing the issue. Ensure your Adobe applications are open before sending commands related to them. If Docker is not running, the tool will alert you in the chat interface.

### Is the software free?
Yes. The software is available for free. Check the GitHub repository for updates and new features.
